五、使用dict

1、Python内置了字典：dict的支持，dict全称dictionary，在其他语言中也称为map，使用键-值（key-value）存储，具有极快的查找速度。

2、如果要删除dict中的一个key，可以用pop(key)方法，对应的value也会从dict中删除。

3、和list比较，dict有以下几个特点：

  • 查找和插入的速度极快，不会随着key的增加而变慢；
  • 需要占用大量的内存，内存浪费多。

而list相反：

  • 查找和插入的时间随着元素的增加而增加；
  • 占用空间小，浪费内存很少。

所以，dict是用空间来换取时间的一种方法。
